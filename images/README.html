📸 关于网站图片

提示： 本网站需要添加图片才能完整显示。请按照以下说明添加图片。

需要的图片

请在 images 文件夹中添加以下图片：

  • avatar.jpg - 个人头像（推荐尺寸：500x500px，正方形）
  • blog1.jpg - 第一篇博客封面图（推荐尺寸：800x600px）
  • blog2.jpg - 第二篇博客封面图（推荐尺寸：800x600px）
  • blog3.jpg - 第三篇博客封面图（推荐尺寸：800x600px）

如何获取图片

你可以从以下网站获取免费的高质量图片：

  • Unsplash - 免费高清图片
  • Pexels - 免费摄影图片
  • Pixabay - 免费图片和视频

图片优化建议

  • 使用 JPG 格式以减小文件大小
  • 头像图片控制在 200KB 以内
  • 博客封面图控制在 500KB 以内
  • 可使用在线工具压缩图片（如 TinyPNG）
